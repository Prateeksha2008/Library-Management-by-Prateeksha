# 📚 Library Management System

A **Library Management System** is a web-based application designed to efficiently manage library operations such as maintaining records of books, users, and transactions like issuing and returning books.

---

## 🚀 Features

* 🔐 User Authentication (Login / Logout)
* 📖 Add, Update, Delete Books
* 👤 User Management
* 📕 Issue Books
* 📗 Return Books
* 🔍 Search Books
* 📊 View Available and Issued Books

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JSP
* **Backend:** Java (Servlets)
* **Database:** MySQL
* **Server:** Apache Tomcat
* **IDE:** Eclipse

---

## ⚙️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/Prateeksha2008/Library-Management-by-Prateeksha.git
   ```

2. Import the project into Eclipse

3. Configure Apache Tomcat Server

4. Set up MySQL Database:

   * Create database
   * Import required tables

5. Run the project on server
